# base-vue

## 安装yarn

```
npm install -g yarn
```

## 安装项目依赖
```
yarn install
```

### 设置配置文件 复制 .env.example 改名为 .env 完善内部配置

### 启动项目开发环境
```
yarn serve
```

### 构建打包项目
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
